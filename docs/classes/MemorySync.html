    Overview

    Backbone Sync for in-memory models.

    Examples:

    How to configure using a model name 

    class Thing extends Backbone.Model
  model_name: 'Thing'
  sync: require('backbone-orm').sync(Thing)

    How to configure using a url 

    class Thing extends Backbone.Model
  url: '/things'
  sync: require('backbone-orm').sync(Thing)
